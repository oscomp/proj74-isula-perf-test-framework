# proj74-isula-perf-test-framework

为iSulad 设计并开发性能测试框架

### **描述**    

支持iSula性能测试的，执行、结果数据处理以及结果展示；

​          支持多种类型的性能测试：包括CRI和Client两种接口类型的性能测试；

​          支持多种容器引擎的性能测试对比，例如iSula和docker、cri-o等引擎的对比；

​          结果显示友好，清晰；
